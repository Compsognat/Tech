Проект представляет собой приложение-парсер для сбора и анализа информации о компьютерных играх с сайта [StopGame](https://stopgame.ru?utm_source=chatgpt.com).
Программа написана на Python и использует веб-скрапинг для автоматического формирования датасета игр в формате CSV. Основной функционал реализован в файле `Project.py`. 

---

# Описание проекта

Проект предназначен для автоматизированного сбора данных о компьютерных играх из каталога сайта StopGame. Приложение получает информацию со страниц игр, обрабатывает HTML-структуру сайта и сохраняет результаты в структурированный CSV-файл.

Система позволяет:

* автоматически получать список игр;
* собирать характеристики каждой игры;
* сохранять данные для дальнейшего анализа;
* отображать процесс работы через графический интерфейс.

Программа содержит GUI-интерфейс на Tkinter с кнопкой запуска и окном логирования действий пользователя. 

---

# Какие датасеты формируются

В результате работы программы создаётся датасет `games.csv`. 

В таблицу записываются следующие поля:

| Поле      | Описание                 |
| --------- | ------------------------ |
| title     | название игры            |
| rating    | пользовательский рейтинг |
| release   | дата выхода              |
| developer | разработчик              |
| genre     | жанр игры                |

Формирование данных происходит методом парсинга HTML-страниц сайта StopGame.

# Какие показатели можно рассчитать

На основе датасета можно вычислить:

### Основные метрики

* средний рейтинг игр;
* количество игр по жанрам;
* количество игр по разработчикам;
* распределение игр по годам выпуска;
* топ разработчиков по среднему рейтингу;
* популярность жанров.

### Дополнительные аналитические показатели

* динамика выпуска игр по годам;
* корреляция между жанром и рейтингом;
* средний рейтинг отдельных разработчиков;
* доля жанров на рынке игр;
* наиболее активные игровые студии.

Сгруппировать игры по жанрам и оценкам.

# Структура проекта

## Основные компоненты

### 1. Графический интерфейс

Класс `MainWindow` реализует:

* окно программы;
* кнопку запуска;
* систему логирования. 

---

### 2. Модуль запуска парсера

Метод `start_parser()` запускает процесс сбора данных в отдельном потоке. 

---

### 3. Получение ссылок

Метод `get_game_links()`:

* переходит по страницам каталога;
* собирает ссылки на игры;
* ограничивает количество игр. 

---

### 4. Парсинг страниц игр

Метод `parse_game()`:

* извлекает название;
* получает рейтинг;
* определяет жанры;
* получает дату выхода;
* извлекает разработчика. 

---

### 5. Экспорт данных

Результаты сохраняются в CSV-файл `games.csv`. 

---

# Результаты работы приложения

После запуска программа:

1. автоматически открывает страницы каталога игр;
2. собирает ссылки на игровые карточки;
3. переходит на страницу каждой игры;
4. извлекает необходимые данные;
5. отображает процесс работы в окне логов;
6. сохраняет результаты в CSV-файл.
